# Images

Upload image files to this folder, then ask the session to place them into the
corresponding placeholder spots on the site (e.g. the Resort & Hotel
Development page from PR #1).

## How to use

1. Add your image files here (`.jpg`, `.png`, `.webp`, etc.).
2. In the session, point to the file and the placeholder it belongs in, e.g.
   "Use `images/hero-riverfront.jpg` for the Resort page hero."
3. The session will wire the image into `index.html` at the matching location.

## Tips

- Use descriptive, lowercase, hyphenated file names (e.g.
  `brewery-exterior.jpg`) so it's clear where each image goes.
- Prefer web-optimized formats and reasonable file sizes for fast page loads.
